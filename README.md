# Братик рабочий бит торрент клиент 
## 1. Сборка проекта(чтобы получить exe)
  1. Открываешь решение в Visual Studio нажимаешь Сборка - Собрать BITtorrent
  2. В /BITtorrent/Release/ лежит собранный exe, который работает на удивление
  3. Пользуешься показываешь учителю какой ты умничка
## 2. Использование приложения
  1. Нажимаешь Add torrent чтобы добавить torrent, выбираешь .torrent файл
  2. Запускаешь скачивание кнопкой Start
  3. Результаты скачивания можешь увидеть в директории которую ты выбрал
  4. Pause - поставить на паузу(с сохранением прогресса)
  5. Remove - удалить torrent из списка и удалить его скачанные файлы
