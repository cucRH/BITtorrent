# Братик рабочий бит торрент клиент 
## 1. Сборка проекта(чтобы получить exe)
  1.1 Открываешь решение в Visual Studio нажимаешь Сборка - Собрать BITtorrent
  1.2 В /BITtorrent/Release/ лежит собранный exe, который работает на удивление
  1.3 Пользуешься показываешь учителю какой ты умничка
## 2. Использование приложения
  2.1 Нажимаешь Add torrent чтобы добавить torrent, выбираешь .torrent файл
  2.2 Запускаешь скачивание кнопкой Start
  2.3 Результаты скачивания можешь увидеть в директории которую ты выбрал
  2.4 Pause - поставить на паузу(с сохранением прогресса)
  2.5 Remove - удалить torrent из списка и удалить его скачанные файлы
