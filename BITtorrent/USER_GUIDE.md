## Введение

BitTorrent Client - это полнофункциональный, удобный и производительный клиент для загрузки торрент-файлов. Приложение разработано с использованием современных технологий C++ и Win32 API, обеспечивая стабильную и быструю работу на всех версиях Windows.

### Требования
- **ОС:** Windows 7 и выше
- **Процессор:** 1 ГГц
- **ОЗУ:** 512 МБ минимум, 2 ГБ рекомендуется
- **Место на диске:** 50 МБ для самого приложения

### Особенности
- ✅ Интуитивный и красивый интерфейс
- ✅ Высокая скорость загрузки
- ✅ Управление полосой пропускания
- ✅ Расписание автоматических загрузок
- ✅ Фильтрация IP адресов
- ✅ Полная поддержка многофайловых торрентов
- ✅ Система уведомлений
- ✅ Мониторинг производительности
- ✅ Горячие клавиши
- ✅ Восстановление состояния после перезагрузки

## Установка

### Способ 1: Автоматическая установка
1. Скачайте установщик `BitTorrentClient-Installer.exe`
2. Запустите установщик
3. Следуйте указаниям мастера установки
4. Выберите папку установки (рекомендуется `C:\Program Files\BitTorrentClient`)
5. Нажмите "Установить"

### Способ 2: Портативная версия
1. Распакуйте архив `BitTorrentClient-Portable.zip` в нужную папку
2. Запустите `BitTorrentClient.exe`

### Первый запуск
При первом запуске приложение:
- Создаст необходимые папки конфигурации
- Инициализирует базу данных
- Установит стандартные горячие клавиши
- Попросит выбрать папку для сохранения загрузок

## Первый запуск

### Начальная конфигурация

#### Шаг 1: Выбор папки загрузок
```
[Выберите папку для сохранения загрузок]
[ Обзор... ]  [ Использовать default ]
```
Рекомендуется выбрать папку на быстром диске.

#### Шаг 2: Предпочтения сети
- **Максимальное количество пиров:** 200 (рекомендуется)
- **Максимум соединений на торрент:** 50
- **Порт прослушивания:** 6881-6889

#### Шаг 3: Параметры интерфейса
- **Тема:** Светлая / Темная
- **Язык:** Русский / English
- **Автозагрузка при старте Windows:** Вкл / Выкл

## Основные функции

### 1. Добавление торрента

#### Способ 1: Через меню
```
File → Open Torrent... (Ctrl+O)
```

#### Способ 2: Перетаскивание
Просто перетащите файл `.torrent` в окно приложения

#### Способ 3: Magnet-ссылка
```
File → Open Magnet Link... (Ctrl+M)
Вставьте magnet-ссылку
```

### 2. Управление загрузками

#### Начало загрузки
1. Выберите торрент в списке
2. Нажмите кнопку "Start" или нажмите Пробел
3. Выберите файлы для загрузки (если требуется)
4. Нажмите "Start Download"

#### Пауза
1. Выберите активную загрузку
2. Нажмите "Pause" или нажмите Пробел

#### Остановка
1. Выберите загрузку
2. Нажмите кнопку "Stop"
3. Выберите: "Keep torrent" или "Delete torrent"

#### Удаление
1. Выберите загрузку
2. Нажмите Delete или правой кнопкой → Remove

### 3. Просмотр информации о загрузке

**Статистика:** (вкладка "Stats")
- Скорость загрузки
- Скорость выгрузки
- Осталось времени
- Количество пиров
- Соотношение

**Информация о файлах:** (вкладка "Files")
- Список файлов в торренте
- Размер каждого файла
- Процент загрузки
- Приоритет загрузки

**Свойства:** (вкладка "Properties")
- Имя торрента
- Размер
- Дата добавления
- Отслеживатель

## Расширенные функции

### 1. Управление полосой пропускания

#### Ограничение скорости
```
Settings → Bandwidth → Speed Limits
├─ Download Limit: 2048 KB/s
├─ Upload Limit: 512 KB/s
└─ Apply Limits: [ Enable ]
```

#### Расписание полосы пропускания
```
Settings → Bandwidth → Schedule
├─ Monday-Friday
│   ├─ 09:00-17:00: 512 KB/s (рабочее время)
│   └─ 17:00-23:00: 2048 KB/s (вечер)
└─ Weekend
	└─ 00:00-23:00: Unlimited
```

### 2. Расписание загрузок

#### Создание расписания
```
File → Schedule Download
├─ Torrent: [Select torrent...]
├─ Time: HH:MM [02:00]
├─ Repeat:
│   ├─ Never (одноразово)
│   ├─ Daily (каждый день)
│   ├─ Weekly (каждую неделю)
│   └─ Monthly (каждый месяц)
└─ [ Add Schedule ]
```

#### Пример
Загружать Linux ISO каждый день в 02:00:
```
1. File → Schedule Download
2. Выберите ubuntu.iso.torrent
3. Установите время: 02:00
4. Выберите Repeat: Daily
5. Нажмите "Add Schedule"
```

### 3. Фильтрация IP адресов

#### Блокирование IP
```
Settings → IP Filter
├─ Blocked IPs:
│   ├─ 192.168.1.0/24 - Block
│   ├─ 10.0.0.0/8 - Block
│   └─ ...
└─ [ Add Rule ] [ Edit ] [ Remove ]
```

#### Предустановленные фильтры
```
Settings → IP Filter → Presets
├─ [ Block Private IPs ]
├─ [ Block ISP ] (требует обновления БД)
└─ [ Block China ] (требует обновления БД)
```

## Советы и рекомендации

### Для максимальной скорости
1. ✓ Установите максимум пиров на 200
2. ✓ Используйте соотношение upload:download 1:4
3. ✓ Держите приложение в фоне без других нагрузок
4. ✓ Отключите VPN если не нужен
5. ✓ Используйте порты выше 10000

### Для стабильности
1. ✓ Регулярно очищайте кеш (Settings → Maintenance → Clear Cache)
2. ✓ Ограничьте максимум соединений (50-100)
3. ✓ Обновляйте список трекеров
4. ✓ Проверяйте здоровье приложения еженедельно

### Для безопасности
1. ✓ Включите фильтрацию IP адресов
2. ✓ Блокируйте приватные IP адреса
3. ✓ Используйте VPN для чувствительных загрузок
4. ✓ Не загружайте из неизвестных источников
5. ✓ Проверяйте хеши файлов

### Для удобства
1. ✓ Используйте горячие клавиши
2. ✓ Включите автозагрузку приложения
3. ✓ Установите расписание загрузок
4. ✓ Используйте недавние файлы (Recent Files)
5. ✓ Персонализируйте интерфейс

## Решение проблем

### Проблема: Низкая скорость загрузки

**Причины и решения:**
```
1. Мало пиров?
   → Убедитесь, что торрент еще популярен
   → Проверьте соединение с трекером

2. Ограничение полосы пропускания?
   → Settings → Bandwidth → Speed Limits
   → Увеличьте лимит

3. Много соединений?
   → Settings → Network → Max Peers: 50-100

4. Проблема с ISP?
   → Используйте VPN
   → Попробуйте порты выше 10000
```

### Проблема: Приложение зависает

**Решение:**
```
1. Нажмите Ctrl+Alt+Delete
2. Откройте Task Manager
3. Найдите BitTorrentClient.exe
4. Нажмите "End Task"
5. Перезагрузитесь и запустите заново

Или используйте:
File → Exit (Ctrl+Q)
```

### Проблема: Ошибка при открытии торрента

**Возможные ошибки:**
```
ERROR: Invalid torrent file
→ Торрент-файл поврежден
→ Скачайте заново

ERROR: Cannot write to disk
→ Недостаточно места на диске
→ Освободите место или измените папку

ERROR: Tracker unavailable
→ Трекер недоступен
→ Попробуйте позже или используйте альтернативные трекеры
```

### Проблема: Порты заняты

**Решение:**
```
Settings → Network → Listening Port
Измените порт с 6881 на другой (например, 12345)
```

### Проблема: Восстановление после сбоя

**Автоматическое восстановление:**
```
При следующем запуске приложение автоматически:
1. Восстановит список загрузок
2. Восстановит прогресс каждой загрузки
3. Восстановит параметры (если они были сохранены)
4. Продолжит прерванные загрузки
```

**Ручное восстановление:**
```
File → Recent Files
Выберите торренты которые были загружены
или
File → Open Recent Database
```
