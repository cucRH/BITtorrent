# BitTorrent Client

Полнофункциональный клиент BitTorrent, написанный на C++ с использованием Win32 API.

## Возможности

### Основная функциональность
- ✅ Загрузка и скачивание торрент-файлов
- ✅ Поддержка мультифайловых торрентов
- ✅ Управление скоростью загрузки/выгрузки
- ✅ Работа с трекерами и пирами
- ✅ Кеширование и управление памятью
- ✅ История загрузок

### UI Компоненты
- 📊 Список активных загрузок с детальной информацией
- 📈 Показатель прогресса загрузки
- 🔢 Отображение скорости загрузки/выгрузки
- ⏱️ Рассчитанное время завершения
- 🌱 Информация о сидам и пирах
- 💾 Управление сохранением файлов

### Технические компоненты

#### TorrentCore.h/cpp
- Парсинг .torrent файлов
- Работа с Bencode форматом
- Управление информацией о кусках данных

#### DownloadManager.h/cpp
- Управление процессом загрузки
- Отслеживание прогресса
- Расчет скорости загрузки/выгрузки
- Поддержка паузы/возобновления

#### TrackerManager.h/cpp
- Взаимодействие с трекерами
- Получение списка пиров
- Управление сидами и личами

#### StorageManager.h/cpp
- Сохранение метаданных торрентов
- История загрузок
- Управление путями хранилища

#### CacheManager.h/cpp
- Кеширование кусков данных
- Управление памятью
- Верификация целостности кусков
- Управление блоками внутри кусков

#### SettingsManager.h/cpp
- Сохранение и загрузка настроек
- Управление портом прослушивания
- Контроль скоростей
- Управление размером кеша

#### Logger.h/cpp
- Логирование событий приложения
- Разные уровни логирования (DEBUG, INFO, WARNING, ERROR, CRITICAL)
- Сохранение логов в файл

#### BitTorrentClient.h/cpp
- Главный класс приложения
- Координация всех компонентов
- Управление активными торрентами

#### UIManager.h/cpp
- Управление пользовательским интерфейсом
- Создание и обновление элементов управления
- Обработка пользовательского ввода

## Использование

### Запуск приложения
```cpp
// Приложение автоматически инициализируется при запуске
// Все настройки загружаются из ./BitTorrent_Data/config.ini
```

### Добавление торрента
1. Нажмите кнопку "Add Torrent"
2. Выберите .torrent файл
3. Нажмите "Start" для начала загрузки

### Управление загрузками
- **Start** - Начать загрузку
- **Pause** - Приостановить загрузку
- **Remove** - Удалить из списка

## Требования

- Windows Vista и выше
- Visual Studio 2017 или новее
- C++11 и выше

## Компиляция

```bash
msbuild BITtorrent.vcxproj /p:Configuration=Release /p:Platform=x64
```

## Настройки

Основные настройки хранятся в `./BitTorrent_Data/config.ini`:

```ini
DownloadPath = ./Downloads
MaxConnections = 100
MaxUploadSpeed = 0
MaxDownloadSpeed = 0
MaxCacheSize = 512
EnableUpload = true
EnableDHT = true
EnablePEX = true
ListeningPort = 6881
Language = English
```
