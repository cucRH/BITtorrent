### Требования
- **Язык:** C++11 и выше
- **Компилятор:** MSVC (Visual Studio 2015+)
- **ОС:** Windows 7 и выше
- **API:** Win32 API
- **Архитектура:** x86 / x64

### Производительность
- **Время запуска:** < 1 сек
- **Память:** 30-50 MB на состояние
