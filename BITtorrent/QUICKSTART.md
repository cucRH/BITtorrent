# 🚀 BitTorrent Client - Быстрый старт

## За 5 минут до первой загрузки

### 1️⃣ Установка (1 минута)

#### Option A: Готовый exe
```powershell
# Просто запустите BITtorrent.exe
.\BITtorrent.exe
```

#### Option B: Из исходников
```powershell
# Откройте Visual Studio
# File → Open → Project/Solution
# Выберите BITtorrent.sln
# Build → Build Solution (Ctrl+Shift+B)
# F5 для запуска
```

### 2️⃣ Первый запуск (2 минуты)

```
1. Приложение запускается
2. Спрашивает папку для загрузок → Выберите или используйте default
3. OK → Готово!
```

### 3️⃣ Первая загрузка (2 минуты)

#### Способ 1: Файл
```
1. File → Open Torrent... (Ctrl+O)
2. Выберите .torrent файл
3. Выберите файлы для загрузки
4. Нажмите "Start"
```

#### Способ 2: Drag & Drop
```
1. Перетащите .torrent в окно приложения
2. Выберите файлы
3. Нажмите "Start"
```

## ⚙️ Базовые параметры

### 1. Папка загрузок
```
Settings → Paths → Download folder
[Выбрать] → Выберите папку
```

### 2. Скорость
```
Settings → Bandwidth → Speed Limits
Download:  2048 KB/s (рекомендуется)
Upload:     512 KB/s
```

### 3. Максимум пиров
```
Settings → Network → Max peers
200 (для большинства торрентов)
```

5. ✗ Не загружайте подозрительные файлы

## 🔍 Часто встречаемые вопросы

### В: Почему медленная скорость?
**A:**
```
Проверьте:
1. Количество пиров (должно быть > 10)
2. Ограничение скорости в Settings
3. Интернет соединение (используйте Speedtest)
4. Выбран ли популярный торрент
```

### В: Как восстановить загрузку?
**A:**
```
Приложение автоматически восстановит все загрузки
при перезагрузке. Просто запустите снова.

Или используйте: File → Recent Files
```

### В: Можно ли загружать на несколько дисков?
**A:**
```
Нет, все файлы загружаются в одну папку.
Но вы можете:
1. Загрузить в одну папку
2. Затем переместить файлы вручную
3. Использовать shortcuts/symbolic links
```

### В: Как изменить папку загрузок?
**A:**
```
Settings → Paths → Download folder
[Выбрать] → Выберите новую папку

⚠️ Уже загруженные файлы не переместятся автоматически
```

### В: Приложение зависло. Что делать?
**A:**
```
1. Ctrl+Alt+Delete → Task Manager
2. Найдите BitTorrentClient
3. End Task
4. Перезагрузитесь
```
